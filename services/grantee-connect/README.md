# grantee-connect — Documentation

Grantee self-service portal for progress reporting and disbursement tracking. This /docs folder syncs to docs-central via Doc Mirror.

## Files
- `architecture.md` — Architecture
- `dependencies.yaml` — Dependencies  
- `runbook.md` — Operational runbook

## Overview
grantee-connect is the post-award interface for grant recipients. Grantees submit progress reports, upload deliverable documents, view disbursement schedules, request budget amendments, and track payment status. Serves ~3,800 active grantees across all award programs.
