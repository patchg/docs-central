# grants-portal — Documentation

Public-facing grant application portal. This /docs folder syncs to docs-central via Doc Mirror.

## Files
- `architecture.md` — System architecture and design decisions
- `dependencies.yaml` — External service and database dependencies
- `runbook.md` — Operational runbook and incident procedures

## Overview
grants-portal is the primary intake system for grant applications. Applicants create accounts, complete eligibility screening, upload required documents, and submit formal applications. The system serves ~12,000 active applicants across 40+ active grant programs.
