# review-engine — Documentation

Internal grant review and scoring platform. This /docs folder syncs to docs-central via Doc Mirror.

## Files
- `architecture.md` — System architecture
- `dependencies.yaml` — Dependencies
- `runbook.md` — Operational runbook

## Overview
review-engine manages the peer-review lifecycle for grant applications. Program officers assign applications to review panels; reviewers score applications against rubrics; panel chairs reconcile scores and produce funding recommendations. Integrates directly with grants_core_db to read applications and write scores.
